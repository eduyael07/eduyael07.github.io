Clase de Propiedad `display` en CSS

Esta clase cubrirá los valores fundamentales de la propiedad display: block, inline, inline-block, y los sistemas de layout modernos flex y grid.

display: block (Elementos de Bloque)

Los elementos con display: block tienen las siguientes características:

  • Ocupan todo el ancho disponible por defecto, creando un "bloque".
  • Comienzan en una nueva línea.
  • Respetan las propiedades width, height, margin (vertical y horizontal) y padding.
  • Ejemplos comunes: <div>, <p>, <h1>-<h6>, <ul>, <li>, <section>.

Ejemplo Práctico: display: block

Soy otro bloque (width: auto, height: 70px)

display: inline (Elementos en Línea)

Los elementos con display: inline tienen las siguientes características:

  • Ocupan solo el ancho necesario para su contenido.
  • No comienzan en una nueva línea, fluyen con el texto circundante.
  • No respetan width ni height.
  • Respetan padding y margin horizontales (left/right).
  • padding y margin verticales (top/bottom) pueden tener efectos visuales, pero no afectan el layout de otros elementos (pueden superponerse).
  • Ejemplos comunes: <span>, <a>, <img> (aunque img es un caso especial, se comporta más como inline-block), <strong>, <em>.

display: inline-block (Bloque en Línea)

Combina características de block e inline:

  • Se colocan en la misma línea que otros elementos (como inline).
  • Respetan width, height, margin (vertical y horizontal) y padding (como block).
  • El ancho por defecto es el necesario para su contenido (como inline), a menos que se especifique un width.
  • Es muy útil para crear elementos de navegación, botones, o pequeñas cajas que deben estar una al lado de la otra y tener dimensiones específicas.
  • Una cosa a tener en cuenta es el espacio en blanco en el HTML entre elementos inline-block, que se renderiza como un espacio. Se puede solucionar con varios trucos (ej. font-size: 0 en el padre).

Ejemplo Práctico: display: inline-block

Para Probar (Ejercicios con display: inline-block):

  1. Cambia el width o height de una de las cajas.
  2. Añade margin-left: 20px; a la segunda caja.
  3. Cambia vertical-align en .inline-block-example .box a middle o bottom.
  4. Añade más cajas. Observa cómo pasan a la siguiente línea si no caben.

Resumen rápido hasta ahora:

  • block: Nueva línea, ancho completo, respeta W/H/M/P.
  • inline: Misma línea, ancho de contenido, NO respeta W/H, M/P verticales "raros".
  • inline-block: Misma línea, respeta W/H/M/P. ¡Lo mejor de dos mundos para ciertos casos!

display: flex (Flexbox)

Flexbox es un modelo de layout unidimensional diseñado para distribuir espacio entre ítems en un contenedor, incluso cuando sus tamaños son desconocidos o dinámicos. Se activa aplicando display: flex al contenedor padre.

Propiedades del Contenedor Flex (Padre)

  • flex-direction: Define el eje principal (row, row-reverse, column, column-reverse).
  • justify-content: Alinea los ítems en el eje principal (flex-start, flex-end, center, space-between, space-around, space-evenly).
  • align-items: Alinea los ítems en el eje secundario (perpendicular al principal) (flex-start, flex-end, center, stretch, baseline).
  • flex-wrap: Controla si los ítems deben envolverse en múltiples líneas (nowrap, wrap, wrap-reverse).
  • align-content: Similar a justify-content, pero para alinear las líneas de ítems cuando hay flex-wrap: wrap y espacio extra en el eje secundario.
  • gap, row-gap, column-gap: Define el espacio entre los ítems.

Propiedades de los Ítems Flex (Hijos)

  • flex-grow: Factor de crecimiento de un ítem si hay espacio extra.
  • flex-shrink: Factor de encogimiento si no hay suficiente espacio.
  • flex-basis: Tamaño inicial del ítem antes de distribuir el espacio.
  • flex: Atajo para flex-grow, flex-shrink y flex-basis.
  • order: Orden visual de los ítems.
  • align-self: Sobrescribe align-items para un ítem individual.

display: grid (CSS Grid Layout)

CSS Grid es un sistema de layout bidimensional que permite organizar el contenido en filas y columnas. Es muy potente para layouts complejos de página.

Propiedades del Contenedor Grid (Padre)

  • grid-template-columns: Define las columnas de la cuadrícula (ej: 1fr 1fr 1fr, 100px auto 200px, repeat(3, 1fr)).
  • grid-template-rows: Define las filas de la cuadrícula (ej: 100px 200px, minmax(100px, auto)).
  • gap (o grid-gap): Espacio entre celdas (atajo para row-gap y column-gap).
  • grid-template-areas: Define una plantilla nombrando áreas de la cuadrícula.
  • justify-items: Alinea el contenido DENTRO de las celdas del grid en el eje de fila (horizontal). (start, end, center, stretch).
  • align-items: Alinea el contenido DENTRO de las celdas del grid en el eje de columna (vertical). (start, end, center, stretch).
  • justify-content: Alinea la cuadrícula completa dentro de su contenedor en el eje de fila (si la cuadrícula es más pequeña que el contenedor).
  • align-content: Alinea la cuadrícula completa dentro de su contenedor en el eje de columna (si la cuadrícula es más pequeña que el contenedor y tiene altura definida).

Propiedades de los Ítems Grid (Hijos)

  • grid-column-start / grid-column-end: Dónde empieza/termina un ítem en las columnas.
  • grid-row-start / grid-row-end: Dónde empieza/termina un ítem en las filas.
  • grid-column: Atajo para grid-column-start / grid-column-end.
  • grid-row: Atajo para grid-row-start / grid-row-end.
  • grid-area: Asigna un ítem a un área nombrada (definida con grid-template-areas) o es un atajo para las 4 propiedades anteriores.
  • justify-self: Sobrescribe justify-items para un ítem individual.
  • align-self: Sobrescribe align-items para un ítem individual.

Conclusión y Próximos Pasos

¡Felicidades! Has explorado los fundamentos de block, inline, inline-block, y los potentes sistemas de layout flexbox y grid.

La clave es la práctica:

  • Intenta recrear layouts de páginas web sencillas.
  • Combina flexbox y grid: grid para el layout general de la página, flexbox para componentes internos.
  • Consulta recursos como MDN Web Docs, CSS-Tricks (Flexbox Guide, Grid Guide) para profundizar.
  • Juega con herramientas interactivas online como Flexbox Froggy o Grid Garden.

Entender cómo se muestran y posicionan los elementos es crucial para construir cualquier interfaz web. ¡Sigue experimentando!
